# Choix De La Technologie

## Contexte

Dans le cadre du projet Techno Boost, l'objectif etait de choisir une technologie a approfondir individuellement, puis de realiser une application concrete permettant de demontrer sa maitrise.

## Technologies Reperees

Lors de la phase de veille, plusieurs technologies ont ete identifiees comme frequemment demandees dans les offres d'emploi et adaptees au metier de developpeur :

- .NET
- C#
- WPF
- SQL / SQLite
- ASP.NET Core
- Angular
- React
- Python
- Django
- Java

## Trois Technologies Retenues Pour Etude

Parmi cette premiere liste, trois technologies ont ete retenues pour un examen plus approfondi :

1. .NET / C#
2. WPF
3. ASP.NET Core

## Analyse Rapide Des 3 Choix

### 1. .NET / C#

- tres present dans les offres d'emploi
- ecosyteme complet pour les applications desktop, web et services
- adapte a un projet structure avec base de donnees et logique metier

### 2. WPF

- technologie desktop adaptee aux applications de gestion
- permet de construire une interface riche sur Windows
- bon support de l'architecture applicative en C#

### 3. ASP.NET Core

- tres utilise pour le developpement web moderne en .NET
- interessant pour les API et applications web
- plus large a mettre en place pour un projet individuel avec interface complete dans un temps court

## Technologie Finalement Choisie

La technologie principale retenue pour ce projet est :

## .NET avec C# et WPF

## Justification Du Choix

Ce choix a ete fait pour les raisons suivantes :

- la stack est coherente pour realiser une application de gestion de taches complete
- elle permet de travailler a la fois l'interface utilisateur, la logique metier et la persistance des donnees
- WPF est une technologie pertinente pour un projet desktop sur Windows
- .NET et C# sont des technologies solides et recherchées dans le monde professionnel
- le projet permet de demontrer des competences concretes sur une application exploitable

## Technologies Effectivement Utilisees Dans Le Projet

Le projet final mobilise les technologies suivantes :

- .NET 10
- C#
- WPF
- Entity Framework Core
- SQLite

## Validation Pedagogique

Sujet retenu pour approfondissement : developpement d'une application de gestion de taches en technologie .NET / WPF.

Cette application sert de support pour demontrer l'apprentissage de la technologie choisie a travers un cas concret de type application de gestion.
